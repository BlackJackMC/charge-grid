{\footnotesize\ttfamily charge\+\_\+grid.\+genetic\+\_\+algorithm.\+main (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



The primary execution script for running the optimization experiments. 

This function sets up and executes a series of genetic algorithm runs. It iterates over defined lists of crossover and mutation operators, dynamically building a configuration dictionary for each combination. It then initializes an Experiment object with this configuration and triggers the optimization process.

\begin{DoxyParagraph}{Function Connections\+:}

\begin{DoxyItemize}
\item Utilizes the globally parsed {\ttfamily data\+\_\+tuple} loaded by {\ttfamily read\+\_\+input} (from {\ttfamily \doxylink{utils_8py}{utils.\+py}}).
\item Injects custom genetic operators (e.\+g., {\ttfamily noise\+\_\+injected\+\_\+adaptive\+\_\+mutation} from {\ttfamily \doxylink{utils_8py}{utils.\+py}}) and routing models (e.\+g., {\ttfamily Alternating\+Routing} from {\ttfamily models}) directly into the config.
\item Instantiates the {\ttfamily Experiment} class to prepare the Py\+GAD engine.
\item Calls {\ttfamily exp\+\_\+temp.\+run()} to execute the evolutionary loop and trigger the data export. 
\end{DoxyItemize}
\end{DoxyParagraph}
