Solves the optimal EV station placement problem using a Mixed-\/\+Integer Linear Programming (\doxylink{namespace_m_i_l_p}{MILP}) solver. 

This function provides an exact mathematical baseline for the routing optimization using Google OR-\/\+Tools (SCIP solver). It defines binary decision variables for station placement (\$x\+\_\+j\$) and integer variables for demand flow (\$\+F\+\_\+\{i,j\}\$).

The solver maximizes the objective function\+: \$\$\textbackslash{}text\{Maximize \} \textbackslash{}lambda(E) -\/ O\$\$ Where \$\+E\$ is total profit (revenue -\/ costs) and \$\+O\$ is the combined operational penalty (unmet demand + distance traveled).

It enforces the following strict mathematical constraints\+:
\begin{DoxyItemize}
\item {\bfseries{Demand Limit\+:}} \$\textbackslash{}sum\+\_\+j F\+\_\+\{i,j\} \textbackslash{}le D\+\_\+i\$
\item {\bfseries{Distance Tolerance\+:}} \$\+F\+\_\+\{i,j\} = 0 \textbackslash{}text\{ if \} L\+\_\+\{i,j\} \texorpdfstring{$>$}{>} Z\+\_\+i\$
\item {\bfseries{Station Activation\+:}} \$\+F\+\_\+\{i,j\} \textbackslash{}le D\+\_\+i \textbackslash{}cdot x\+\_\+j\$
\item {\bfseries{Station Capacity\+:}} \$\textbackslash{}sum\+\_\+i F\+\_\+\{i,j\} \textbackslash{}le B \textbackslash{}cdot x\+\_\+j\$
\end{DoxyItemize}
