\doxysubsection{Detailed Description}
The abstract base class for all EV station routing models. 

This class defines the standard interface and shared state initialization for all routing algorithms in the project. It stores the core network matrices and configuration settings so that inheriting classes do not need to duplicate this boilerplate code. It enforces a strict contract by requiring child classes to implement the {\ttfamily route}, {\ttfamily fitness}, and {\ttfamily get\+\_\+details} methods.

\begin{DoxyParagraph}{Class Connections\+:}

\begin{DoxyItemize}
\item Acts as the parent class for specific models like {\ttfamily Alternating\+Routing}, {\ttfamily Customer\+Routing}, and {\ttfamily Station\+Routing}.
\item Instantiated dynamically by {\ttfamily Experiment.\+\_\+\+\_\+init\+\_\+\+\_\+} based on the provided configuration. 
\end{DoxyItemize}
\end{DoxyParagraph}
