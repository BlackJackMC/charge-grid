Prompts the user via CLI to select multiple saved optimization results for comparison. 

Scans the predefined output directory for JSON files matching the \textquotesingle{}solution\+\_\+\texorpdfstring{$\ast$}{*}.json\textquotesingle{} pattern. It displays an indexed list and accepts a string of numbers (comma or space-\/separated) or the keyword \textquotesingle{}all\textquotesingle{} to select multiple files simultaneously. Uses regex to parse the user input.
