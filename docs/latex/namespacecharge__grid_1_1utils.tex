{\footnotesize\ttfamily charge\+\_\+grid.\+utils.\+adaptive\+\_\+mutation (\begin{DoxyParamCaption}\item[{}]{data\+\_\+tuple }\end{DoxyParamCaption})}



A factory function that generates a demand-\/aware adaptive mutation operator. 

This function extracts the customer demand vector {\ttfamily D} from the provided data tuple to create a custom mutation closure for Py\+GAD.

The generated mutation operator iterates through each chromosome in the offspring array. For each chromosome, it performs a coin toss (50\% probability)\+:
\begin{DoxyItemize}
\item Add a station\+: It identifies all empty nodes (0s). It then calculates a probability distribution based on the localized demand at those empty nodes. A new station is added (flipped to 1), heavily favoring locations with high unfulfilled demand.
\item Remove a station\+: It identifies all currently active stations (1s) and drops one (flipped to 0) uniformly at random.
\end{DoxyItemize}

{\footnotesize\ttfamily charge\+\_\+grid.\+utils.\+custom\+\_\+intersection\+\_\+crossover (\begin{DoxyParamCaption}\item[{}]{data\+\_\+tuple }\end{DoxyParamCaption})}



A factory function that generates a spatially-\/aware custom crossover operator. 

This function acts as a closure, extracting the distance matrix {\ttfamily L} from the provided data tuple to be used during the crossover phase. It returns a custom crossover function formatted for Py\+GAD.

The generated crossover algorithm uses a dynamic clustering approach to preserve geographic structures. For each pair of parents, it randomly selects up to {\ttfamily K} (10) medoid nodes and groups all other nodes into clusters based on their proximity in the distance matrix {\ttfamily L}. Offspring then inherit entire geographical clusters of genes from either the first or second parent with a 50\% probability, preventing the fragmentation of highly optimized local station networks.

A factory function generating a state-\/aware mutation operator with random noise injection. 

This function builds upon the {\ttfamily stagnation\+\_\+aware\+\_\+adaptive\+\_\+mutation} logic by introducing a baseline probability of completely random bit-\/flipping.

For each chromosome in the offspring array\+:
\begin{DoxyEnumerate}
\item {\bfseries{Noise Injection\+:}} There is a 10\% chance to bypass all adaptive logic entirely and simply flip one random gene (station state). This guarantees a baseline level of genetic diversity to prevent premature convergence.
\item {\bfseries{Stagnation Recovery\+:}} If Py\+GAD detects stagnation (\texorpdfstring{$>$}{>}50 generations without improvement), it intelligently drops 20\% of the active stations, heavily targeting geographically redundant clusters.
\item {\bfseries{Dynamic Normal Operation\+:}} If neither of the above apply, it scales the probability of adding vs. dropping a station based on the current network utilization (Demand / Capacity). When adding, it favors high-\/demand nodes; when dropping, it targets redundant nodes.
\end{DoxyEnumerate}

Computes the operational penalty metrics for unmet demand and travel distance. 

Evaluates the operational inefficiency and customer dissatisfaction of the current routing assignment. It calculates the total unfulfilled demand by subtracting the total served flow from the initial demand vector {\ttfamily D}. It also calculates the total distance traveled by multiplying the flow matrix {\ttfamily F} by the distance matrix {\ttfamily L}. These raw values are scaled by their respective penalty weights ({\ttfamily alpha} and {\ttfamily beta}) to produce a final, combined penalty score.

A factory function generating an advanced, state-\/aware adaptive mutation operator. 

This function extracts demand ({\ttfamily D}), distances ({\ttfamily L}), and station capacity ({\ttfamily B}) from the data tuple to construct a highly dynamic Py\+GAD mutation operator. The operator adjusts its behavior based on the current state of the genetic algorithm and the specific characteristics of each chromosome\+:


\begin{DoxyEnumerate}
\item {\bfseries{Stagnation Recovery\+:}} If Py\+GAD detects no fitness improvement for over 50 generations, the operator forces a major structural shift to escape local optima. It identifies active stations that are geographically too close to one another (redundant) and simultaneously drops 20\% of them.
\item {\bfseries{Dynamic Normal Operation\+:}} If not stagnant, the operator calculates the network\textquotesingle{}s current utilization (Total Demand / Total Network Capacity).
\begin{DoxyItemize}
\item High utilization increases the probability of adding a new station (favoring high-\/demand nodes).
\item Low utilization increases the probability of dropping a station (specifically targeting redundant stations clustered too closely together).
\end{DoxyItemize}
