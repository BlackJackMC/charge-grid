{\footnotesize\ttfamily charge\+\_\+grid.\+visualizer.\+build\+\_\+map.\+generate\+\_\+interactive\+\_\+map (\begin{DoxyParamCaption}\item[{}]{best\+\_\+x,  }\item[{}]{df\+\_\+meta,  }\item[{}]{N,  }\item[{}]{B,  }\item[{}]{C,  }\item[{}]{P,  }\item[{}]{R,  }\item[{}]{Z,  }\item[{}]{D,  }\item[{}]{L,  }\item[{}]{config,  }\item[{}]{model,  }\item[{}]{shuffle\+\_\+seed = {\ttfamily 42} }\end{DoxyParamCaption})}



Orchestrates the creation and export of the interactive HTML map. 

Recalculates the flow matrix {\ttfamily F} using the dynamically loaded routing model. It computes the overall economic (E) and operational (O) metrics for the UI overlay. It then builds the spatial data structures, retrieves OSRM routes, constructs a base {\ttfamily folium.\+Map}, and injects custom CSS, Java\+Script, and JSON data directly into the HTML DOM before saving the file to the cache directory.

\begin{DoxyParagraph}{Function Connections\+:}

\begin{DoxyItemize}
\item Calls {\ttfamily model.\+route()} or {\ttfamily model.\+solve\+\_\+flow()} to resolve the flow matrix for the saved layout.
\item Calls {\ttfamily E} and {\ttfamily O} (from {\ttfamily \doxylink{utils_8py}{utils.\+py}}) to recalculate total metrics.
\item Calls {\ttfamily build\+\_\+map\+\_\+data} and {\ttfamily append\+\_\+osrm\+\_\+routes} to generate the payload.
\item Reads external assets ({\ttfamily map\+\_\+style.\+css}, {\ttfamily map\+\_\+script.\+js}) and binds them to the HTML.
\end{DoxyItemize}
\end{DoxyParagraph}
