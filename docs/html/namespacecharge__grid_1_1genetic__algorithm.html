charge_grid.genetic_algorithm.main ( )

The primary execution script for running the optimization experiments.

This function sets up and executes a series of genetic algorithm runs. It iterates over defined lists of crossover and mutation operators, dynamically building a configuration dictionary for each combination. It then initializes an Experiment object with this configuration and triggers the optimization process.

Function Connections:
  • Utilizes the globally parsed data_tuple loaded by read_input (from utils.py).
  • Injects custom genetic operators (e.g., noise_injected_adaptive_mutation from utils.py) and routing models (e.g., AlternatingRouting from models) directly into the config.
  • Instantiates the Experiment class to prepare the PyGAD engine.
  • Calls exp_temp.run() to execute the evolutionary loop and trigger the data export.
