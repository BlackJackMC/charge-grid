charge_grid.visualizer.graph_dashboard.choose_solution_files ( )

Prompts the user via CLI to select multiple saved optimization results for comparison.

Scans the predefined output directory for JSON files matching the 'solution_*.json' pattern. It displays an indexed list and accepts a string of numbers (comma or space-separated) or the keyword 'all' to select multiple files simultaneously. Uses regex to parse the user input.

Function Connections:
  • Called by the main execution block.
  • The returned list of file paths is passed directly into load_chart_data.
Returns
A list of pathlib.Path objects representing the selected JSON files. Returns an empty list if no files are found or the input is invalid.

◆ generate_dashboard_html()

charge_grid.visualizer.graph_dashboard.generate_dashboard_html ( chart_data)

Orchestrates the creation and export of the interactive HTML dashboard.

Reads the external CSS (dashboard_style.css) and JavaScript (dashboard_script.js) asset files. It constructs a standalone HTML template that includes the Chart.js library, injects the parsed chart_data directly into a script tag as a JSON string, and saves the final dashboard.html file into the cache directory.

Function Connections:
  • Consumes the data dictionary created by load_chart_data.
  • Called by the main execution block to finalize the export.
Parameters
chart_dataA structured dictionary containing the historical metrics and metadata for all selected experiments.

◆ load_chart_data()

charge_grid.visualizer.graph_dashboard.load_chart_data ( selected_files)

Parses the selected JSON files to extract generation history and configuration metadata.

Iterates through the provided file paths to extract the generation_history (for plotting fitness and profit trends over time) and the metadata.configuration (for displaying the GA hyperparameters like mutation type, crossover, alpha, beta, etc.). It formats this data into a structured dictionary optimized for injection into Chart.js.

Function Connections:
  • Receives the file list generated by choose_solution_files.
  • Called by the main execution block to prepare the payload for generate_dashboard_html.
Parameters
selected_filesA list of pathlib.Path objects pointing to the chosen JSON solutions.
Returns
A dictionary where the keys are filenames and the values are dictionaries containing 'title', 'labels', 'fitness', 'profit', and 'extra_meta'.
